## What I Learned

* Creating a Single Page Application (SPA) using React.
* Implementing client-side routing with React Router DOM.
* Creating static and dynamic routes.
* Using `useParams()` to access URL parameters.
* Fetching data from REST APIs using `fetch()`.
* Rendering API data dynamically in React.
* Organizing code using reusable React components.
* Building a responsive user interface.

---

## Features Completed

* Created Home page.
* Created Shop page to display all products.
* Integrated DummyJSON Products API.
* Displayed products in a responsive grid layout.
* Implemented dynamic routing for Product Details.
* Used `useParams()` to fetch product details by ID.
* Displayed product image, title, description, price, category, and rating.
* Implemented navigation without browser page reload.
* Organized the project using reusable React components.
* Added responsive styling using CSS.

## Challenges Faced

* Understanding React Router DOM routing.
* Implementing dynamic routes with `useParams()`.
* Fetching data from APIs and handling asynchronous operations.

## Outcome

By the end of Sprint 6, I built a React-based Single Page E-Commerce Application where users can browse products, open detailed product pages using dynamic routing, and navigate smoothly without refreshing the browser. The project demonstrates React Router DOM, API integration, reusable components, and responsive UI design.
